# 变压器容量选择计算器 - kVA 选型与 GB/T 17468 R10 国标容量推荐

## 引言

**变压器容量选择**是工厂配电、楼宇电气、设计院出计算书必经的一步。传统做法是翻 GB/T 17468《电力变压器选用导则》或 GB 1094《电力变压器》手册,手算 **S = P / (cosφ × η)** 后再到 R10 优先数系容量表里查最近一档,遇到多设备还要乘同时系数 KΣ,遇到电动机负载还要算启动电流,流程繁琐、容易抄错。本工具把核心公式与国标容量序列封装成网页表单,输入即得推荐容量与选型依据,设计、施工、运维都能直接用。

## 工具功能 - 6 种选型模式

### 单设备选型(最常用)

适用场景:一台电动机 / 一组加热器 / 一台整流器供电。核心公式:**S = P / (cosφ × η)**。典型用户:设计院电气专业、工厂单台大功率设备供电。输入参数最简,工程师手算 1 分钟,工具 1 秒出结果。

```
输入: 设备功率 P / 数量 / 功率因数 cosφ / 效率 η
输出: 需求容量 S(kVA) / 推荐国标容量 S_n / 负载率 β / 选型依据
```

### 多设备选型(同时系数)

适用场景:多台设备共用一台变压器,设备不会同时满载。核心公式:**S = ΣP × KΣ / (cosφ_avg × η)**,其中 cosφ_avg 按计算负荷加权。典型用户:工厂车间配电、小区配电房、商业综合体、医院 ICU。

```
输入: 多设备功率 P_i / 数量 / 各类 cosφ_i / 同时系数 KΣ / 加权公式 K_xi
输出: 需求容量 S(kVA) / 推荐国标容量 S_n / 负载率 β / 选型依据
```

### 电动机负载(启动电流修正)

适用场景:变压器主要给电动机供电,直接启动 / 软启动 / 变频启动。核心公式:**S ≥ K_st × ΣP_motor / (cosφ × η)**,其中 K_st 是启动电流倍数。工具对最不利启动组合取大值,而非累加。典型用户:电机密集车间、给水排水泵站、风机水泵房。

```
输入: 电动机总功率 ΣP_motor / 启动方式(DOL/Y-Δ/软启)/ 启动倍数 K_st / cosφ / η
输出: 启动修正后 S / 推荐国标容量 S_n / 启动压降 U_drop% / 端子电压校验
```

### 负载率优化(经济运行)

适用场景:多台变压器并联 / 长期接近满载运行 / 数据中心 24h 不间断供电。核心公式:**S = P / (β_opt × cosφ × η)**,β_opt 取 0.7~0.85。依据:变压器效率曲线在 β = 0.5~0.7 附近最高,长期满载(β ≈ 1.0)时铜损急剧增加。

```
输入: 平均负荷 P_avg / 目标负载率 β_opt(0.7~0.85)/ cosφ / η
输出: 经济运行容量 S / 推荐国标容量 S_n / 实际负载率 β_actual
```

### R10 国标容量推荐

适用场景:算出需求 kVA 后,需要选国标容量档。核心数据:**GB/T 17468 R10 优先数系** 30 / 50 / 63 / 80 / 100 / 125 / 160 / 200 / 250 / 315 / 400 / 500 / 630 / 800 / 1000 / 1250 / 1600 / 2000 / 2500 kVA 共 19 档。典型用户:所有需要出计算书、采购变压器铭牌的场景。

```
输入: 需求容量 S_demand(kVA)/ 扩展余量档(默认 15%)
输出: R10 容量档推荐 S_n / 实际负载率 β_actual / 是否经济运行区
```

### 与 UPS / 发电机配合

适用场景:变压器下接 UPS,或变压器 + 柴油发电机备用。核心公式:**S_transformer ≥ S_UPS / 0.8**,**S_generator ≈ S_transformer × 1.1~1.25**。考虑 UPS 输入谐波(6 脉冲整流 THDi ≈ 33%),变压器还要降容 10%~20%。典型用户:数据中心、医院、应急供电系统、银行灾备中心。

```
输入: UPS 容量 S_UPS / 整流脉冲数(6/12)/ 发电机阶跃负载系数
输出: 变压器最小容量 S_T / 发电机推荐容量 S_G / 谐波降容后修正值
```

- **1.0** — 满载,无扩展余量(不推荐长期)
- **自填** — 0.5 ~ 1.2 范围

## 输出说明

### 需求容量 S_demand

定义为:根据输入的设备功率、功率因数、效率、同时系数、启动倍数等参数,经计算得到的变压器最小需求容量。计算公式:**S_demand(kVA) = P(kW) × KΣ × K_st / (cosφ × η)**。工程含义:选型时,**国标容量 S_n ≥ S_demand** 才满足要求,这是硬约束。

### 推荐国标容量 S_n

定义为:从 **GB/T 17468 R10 优先数系** 中,选取不小于 S_demand 的最近一档容量。工具不对外暴露原始表格,只把推荐容量作为输出项之一呈现。

**推荐原则**:

- **S_n ≥ 1.15 × S_demand** — 留 15% 扩展余量,**默认推荐**(3 年内负荷增长也顶得住)
- **S_n = 1.0 × S_demand** — 刚好满足,无扩展(短期可接受,长期不推荐)
- **S_n < S_demand** — ⚠️ **告警:容量不足,需选更大一档或并联**

### 实际负载率 β_actual

定义为:**S_demand / S_n**,反映变压器在选型后的实际负载水平。工程含义:

- **0.6 ~ 0.85** — ✅ **经济运行区**(变压器效率最高,空载 / 负载损耗比例最优)
- **0.85 ~ 0.95** — 满载区,短期可接受,长期铜损急剧上升
- **> 0.95** — ⚠️ 过载风险,绝缘老化加速,寿命缩短
- **< 0.5** — ⚠️ 长期低负载,大马拉小车,空载损耗占比高、效率低

### 选型依据

工具不输出"通过 / 不通过"二元判断,而是给出文字提示,例如:

- ✅ "符合 GB/T 17468 R10 容量推荐,实际负载率 78%(经济运行区)"
- ⚠️ "容量偏小,建议选 100 kVA 而非 80 kVA,留扩展余量"
- ⚠️ "实际负载率 92%,接近过载,长期运行不经济"
- 💡 "提示:同步电动机可做无功补偿,实际 cosφ 可调高至 0.95"

### 启动电流校验(电动机场景)

当设备类型为电动机时,工具额外计算:

- **启动视在功率**:S_st = √3 × U × I_st
- **变压器阻抗压降**:U_drop% ≈ S_st / S_n × U_k%(U_k% 典型 4%~6%,GB 1094)
- **端子电压校验**:U_terminal ≥ 0.85 × U_n(GB 50055 电机启动端子电压要求)
- **输出示例**:「启动时母线压降 12%,符合 GB 50055 电机启动端子电压 ≥ 85% 额定要求」

## 公式详解

### 核心公式(单设备)

```
S(kVA) = P(kW) / (cosφ × η)
```

- **P** — 设备有功功率(kW)
- **cosφ** — 功率因数(0~1 之间,典型 0.7~0.98)
- **η** — 变压器效率(典型 0.95~0.98,大容量更高)

**算例**:100 kW 电动机负载,cosφ = 0.85,η = 0.97 → S ≈ 100 / (0.85 × 0.97) ≈ **121 kVA** → 向上选 R10 档 → **125 kVA**。

### 多设备同时系数

```
S(kVA) = KΣ × ΣP_i / (cosφ_avg × η)
```

- **KΣ** — 同时系数(0.6~0.9,见预设值表)
- **ΣP_i** — 第 i 类设备有功功率之和(kW)
- **cosφ_avg** — **加权平均功率因数**

**加权公式**(依据 GB 50054-2011 §3.4.5):

```
cosφ_avg = Σ(P_i × K_xi × cosφ_i) / Σ(P_i × K_xi)
```

其中 P_i = 第 i 类设备总有功功率(kW),K_xi = 需要系数(0.4~1.0),cosφ_i = 第 i 类设备功率因数。**注意**:分子分母都乘 K_xi,即"**按计算负荷加权**"而非"按设备总额定功率加权"——这样才能反映设备不会同时满载的真实运行情况。

### 电动机启动修正

```
S(kVA) ≥ K_st × ΣP_motor / (cosφ × η)
```

- **K_st** — 启动电流倍数
  - **DOL**:5~7(GB 50055 默认值,小容量电机)
  - **Y-Δ**:2~3(降压启动,适用笼型电机)
  - **软启动**:1.0~1.5(平滑启动,大功率首选)
- 公式**对最不利启动组合**取大值,而非累加
- 变压器短路阻抗 U_k% 也会影响启动压降,需要单独校验

### 经济负载率

```
β_opt = 0.7 ~ 0.85   (经济运行区)
S_optimal = P_avg / (β_opt × cosφ × η)
```

**依据**:变压器效率曲线在 β = 0.5~0.7 附近最高,长期满载(β ≈ 1.0)时铜损急剧增加。工程惯例:选型时按 β = 0.85 计算,留 15% 扩展余量。

### 选型校验

```
S_demand ≤ S_n ≤ S_demand × 1.3   (留 15%~30% 扩展余量,推荐)
或
S_demand ≤ S_n                     (刚好满足,无扩展,不推荐)
```

注:**1.3 倍上限是工程惯例**,不写入工具输出,只放教育段落,避免工程师误用。

## 常见问题 (FAQ)

### 变压器容量怎么算最准确?

核心公式 **S(kVA) = P(kW) / (cosφ × η)**,P 是有功功率,cosφ 是功率因数,η 是变压器效率。多设备时 **S = KΣ × ΣP / (cosφ_avg × η)**,KΣ 是同时系数(0.6~0.9)。电动机负载要算启动倍数,**K_st × P_motor** 计算后再除 (cosφ × η),**不能简单累加**。**最准做法**:算出 S 后查 GB/T 17468 R10 容量表,选不小于 S 的最近一档,并留 15%~30% 扩展余量。

### 功率因数 cosφ 对变压器容量有什么影响?

cosφ 越低,所需 kVA 越大,变压器越贵,线路损耗越高。例:100 kW 负载,cosφ = 0.95 → S ≈ 110 kVA;cosφ = 0.7 → S ≈ 149 kVA(**贵 35%**)。**对策**:装电容补偿把 cosφ 提到 0.9 以上,中国《功率因数调整电费办法》也鼓励补偿(低于 0.9 会被罚款)。变压器铭牌标 kVA 不标 kW,就是因为 cosφ 不可控,只能保证最大视在容量 S。

### 变压器负载率多少最合适?

经济运行区是 **0.7~0.85**,不是越高越好。负载率 0.5 以下(大马拉小车):空载损耗占比高,效率低、不经济。负载率 0.85~0.95:接近满载,短期可接受,长期铜损急剧上升。负载率 > 0.95:过载风险,绝缘老化加速,寿命缩短。**工程惯例**:选型时按 β = 0.85 计算,留 15% 扩展余量(三年内负荷增长也顶得住)。

### 变压器效率 η 一般取多少?

配电变压器(10/0.4 kV,容量 100~2500 kVA)效率通常 **0.95~0.98**。小型变压器(30~80 kVA):η ≈ 0.93~0.95。中型变压器(100~800 kVA):η ≈ 0.95~0.97。大型变压器(1000~2500 kVA):η ≈ 0.97~0.98。**新国标 GB 20052-2020** 对 1 级 / 2 级能效变压器的损耗有更严要求,实际采购时认准能效等级比 η 数值更重要。**注意**:η 是变压器自身的效率,不是负载的功率因数,两个概念别混。

### 变压器容量选择依据哪个国标?

首选 **GB/T 17468-2019《电力变压器选用导则》**(替代 GB/T 17468-1998 旧版),容量 R10 序列(30 / 50 / **63** / 80 / 100 / 125 / 160 / 200 / 250 / 315 / 400 / 500 / 630 / 800 / 1000 / 1250 / 1600 / 2000 / 2500 kVA,共 **19 档**)就来自该标准。配套标准:GB 1094《电力变压器》系列(温升、绝缘、短路阻抗)、GB 50052《供配电系统设计规范》(变压器能效限定值)、GB 20052《电力变压器能效等级》。**涉外项目** 可参考 IEC 60076 系列,容量序列与 GB R10 一致。

### 常用变压器容量有哪些?

GB/T 17468 R10 优先数系 **19 档**(单位 kVA):

- **小容量(≤100)**:30 / 50 / **63** / 80 / 100,适合小型车间、工地临时电(**63 kVA** 工商业配电常用中位档)
- **中容量(160~630)**:160 / 200 / 250 / 315 / 400 / 500 / 630,工厂配电、小区配电房主力档位
- **大容量(800~2500)**:800 / 1000 / 1250 / 1600 / 2000 / 2500,区域变电站、大型工厂、商业综合体

**选型原则**:算出 S 后,**向上选一档**,留 15%~30% 扩展余量。算出来 180 kVA 选 200,不选 160(满载会缩短寿命)。

### 变压器与 UPS 容量如何匹配?

经验公式 **S_transformer ≥ S_UPS / 0.8**。例:100 kVA UPS,前端变压器至少选 125 kVA(125 × 0.8 = 100,刚好覆盖 UPS 满载)。考虑 UPS 输入谐波(6 脉冲整流 THDi ≈ 33%),变压器还要降容 10%~20%,**实际选 ≥ 160 kVA**。**与发电机配合**:**S_generator ≈ S_transformer × 1.1~1.25**(考虑发电机带非线性负载降容 + 阶跃性负载能力)。**关键点**:UPS 容量(kVA)≠ 实际负载(kW),cosφ UPS 通常 0.9~1.0(现代 UPS),老式 UPS 可能 0.8。
